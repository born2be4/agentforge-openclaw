# AgentForge — Kim Edition (RU)

Короткая операционная версия AgentForge для прод-контуров OpenClaw.

## Что изменено относительно базовой версии

1. **Minimal notifications policy**  
   Уведомления не «на каждый шаг», а только на вехах: старт большой задачи / блокер / итог.

2. **Safe restart policy**  
   Рабочие агенты не перезапускают gateway самостоятельно. Рестарт делает внешний оператор/координатор.

3. **Профили сложности**  
   - **Lite** (3–5 файлов) — маски и простые роли  
   - **Standard** (8–10 файлов) — регулярная операционка  
   - **Full** (12–13 файлов) — долгие контекстные роли

4. **MemorySearch decision rules**  
   Включается по условиям (длинные диалоги, важна историчность решений), а не автоматически для всех.

## Быстрый выбор профиля

- Нужен простой эксперт в топике → **Lite**
- Нужен рабочий агент с памятью правил/результатов → **Standard**
- Нужен «долгоживущий коллега» с handoff и самоулучшением → **Full**

## Проверка перед продом

- [ ] tools.deny покрывает рискованные инструменты
- [ ] memorySearch включён только при реальной пользе
- [ ] AGENTS.md не требует спам-уведомлений
- [ ] нет инструкций «рестартни gateway изнутри агента»
- [ ] выбран профиль (lite/standard/full) под реальную задачу

---

Эта версия ориентирована на стабильность, низкий шум и предсказуемое поведение в рабочем Telegram-контуре.
